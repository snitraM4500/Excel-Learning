🏷️ Understanding Data Types

Qualitative (Categorical) Data

Information about qualities that fall into categories, not directly described by numbers.

Examples Description
Brands Product types or names
Nationality Country of origin
Professions Job titles or occupations
Key Statistic: With categorical data, we calculate proportions and percentages. For example: =COUNTIF(range, criteria) / COUNTA(range) gives you the proportion of a category.

Quantitative (Numerical) Data

Information described directly by numbers and measurements.

Examples Description
Income Monetary amounts
Age Years lived
Height Physical measurements
Key Statistics: With numerical data, we calculate average, sum, range, min, max. For example: =AVERAGE(range), =SUM(range), =MAX(range)-MIN(range)
🎯 Challenge 1: Calculate Proportions If a dataset has 50 people and 30 prefer Brand A, what proportion prefer it?
Formula: =COUNTIF(range, "Brand A") / COUNTA(range)

📊 Bar Charts

Clustered Bar Chart

Used when comparing values across categories. Order is not important, but values matter.

Type Gen 1 Count Gen 2 Count
Grass 12 9
Fire 12 8
Water 30 18
Bug 13 10
This clustered bar chart shows Pokemon type counts side-by-side. We can easily see that Water-type Pokemon are most common and Fire-type are least common in both generations.

Stacked Bar Chart

Highlights the total contribution for each category by stacking bars on top of each other.

This stacked bar chart shows the combined totals. The total bar length represents all Pokemon of each type across both generations. Gen 1 is shown in blue, Gen 2 in orange.

100% Stacked Bar Chart

Shows the proportion of each data column within each category (all scale to 100%).

This chart answers: "What proportion of each type came from Gen 1 vs Gen 2?" Notice that Water-type Pokemon have a different ratio than Fire-type Pokemon.
🎯 Challenge 2: Find the Sum What is the total count of all Grass-type Pokemon across Gen 1 and Gen 2?
Formula: =SUM(A1:A2)

🥧 Pie & Doughnut Charts

2-D Pie Chart

Used with a single data column to show composition and relative sizes of categories.

Type Count
Grass 12
Fire 12
Water 30
Bug 13
The pie chart shows each type as a slice of the whole. Water-type takes up the largest slice, demonstrating it's the most common. This works best for 2-5 categories.

Doughnut Chart

Similar to pie charts but with a hollow center, often used for multiple data columns.

💡 Tip: Use pie/doughnut charts for parts-of-a-whole comparisons. Use bar charts for comparing values across categories. Use line charts to show trends over time.
🎯 Challenge 3: Calculate Percentage Water-type has 30 Pokemon out of 67 total (12+12+30+13). What percentage is that?
Formula: =(part/whole)*100 or =ROUND((part/whole)*100, 2)

🔧 Essential Excel Formulas

Aggregate Functions

=SUM(A1:A10)

Adds all values in the range

=AVERAGE(A1:A10)

Calculates the mean of values

=MAX(A1:A10)

Finds the largest value

=MIN(A1:A10)

Finds the smallest value

Counting & Conditional Functions

=COUNT(A1:A10)

Counts numeric values only

=COUNTA(A1:A10)

Counts all non-empty cells

=COUNTIF(A1:A10,"Water")

Counts cells matching a condition

=SUMIF(A1:A10,"Water",B1:B10)

Sums values based on a condition
